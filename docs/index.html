What's SkimJS?

SkimJS is a self-hosting compiler for a subset of Scheme (R7RS) that compiles into readable, nice JavaScript.

How?

SkimJS is a very new project, it's just a few weeks old!
It can compile only a very small subset of Scheme, barely enough to self-host.

If you'd like to give it a try (warning: it's early stages):

  • Install skim globally (or do your thing if you know what you're doing) npm i -g skimjs
  • Create a file mycode.ss and write some scheme code.
  • Compile code with skim mycode.ss

Repo: https://github.com/muratg/skim
