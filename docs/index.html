What's SkimJS?

SkimJS is a self-hosting compiler for a subset of Scheme (R7RS) that compiles into readable, nice JavaScript.

How can I try it?

SkimJS is a very new project, it's just a few weeks old!
It can compile only a very small subset of Scheme, barely enough to self-host.

If you'd like to give it a try (warning, it's early stages) try this:
npm i -g skimjs
Compile code with skim filename.skim .

(define (factorial n)
  (cond ((num< n 0) #f)
    ((num<= n 1) 1)
    (else (num* n
                (factorial (num- n 1))))))
        
function factorial(n) {
  return (() => {
    if (n < 0) {
      return false;
    } else if (n <= 1) {
      return 1;
    } else {
      return n * factorial(n - 1);
    }
  })();
}
